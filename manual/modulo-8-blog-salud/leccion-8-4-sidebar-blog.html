Lección 8.4: Sidebar del Blog

Componente Blade con buscador, categorías, recientes, newsletter y populares

🎯 Piensa en esto: Imagina el índice de un libro: te permite buscar temas específicos, ver los capítulos organizados por categorías, y descubrir contenido relacionado. El sidebar es exactamente eso: una guía de navegación que ayuda a los lectores a encontrar más contenido de su interés.
1 Crear el componente y consultar datos

Primero obtenemos los datos que vamos a mostrar: artículos recientes y categorías con artículos.

📍 Archivo: resources/views/components/blog-sidebar.blade.php

@php
use App\Models\Articulo;
use App\Models\Categoria;

$recientes = Articulo::publicados()
    ->recientes()
    ->limit(5)
    ->get();

$categorias = Categoria::conArticulosPublicados()
    ->ordenadas()
    ->get();
@endphp

💡 Los scopes publicados(), recientes() y ordenadas() filtran y ordenan los datos automáticamente.

📝 ESTE ES EL CÓDIGO QUE VAMOS A APRENDER:

📍 Archivo: resources/views/components/blog-sidebar.blade.php

<ul class="space-y-2">
    @foreach($categorias as $categoria)
        <li>
            <a href="{{ route('blog.categoria', $categoria->slug) }}">
                <span 
                    class="w-3 h-3 rounded-full"
                    style="background-color: {{ $categoria->color }}"
                ></span>
                <span>{{ $categoria->nombre }}</span>
                <span>
                    {{ $categoria->articulos()->publicados()->count() }}
                </span>
            </a>
        </li>
    @endforeach
</ul>

🔍 Vamos a entender línea por línea (como si fuera la primera vez):

style="background-color: {{ $categoria->color }}"
→ Usamos el color guardado en la base de datos para crear un círculo identificador visual. Cada categoría tiene su propio color.
route('blog.categoria', $categoria->slug)
→ Generamos la URL usando el slug (URL amigable) de la categoría. Es más bonito y SEO-friendly que usar el ID.
$categoria->articulos()->publicados()->count()
→ Contamos cuántos artículos publicados tiene cada categoría. El número aparece como un badge al lado del nombre.
2 El buscador con formulario

Un campo de búsqueda simple que envía el término a una ruta específica.

📍 Archivo: resources/views/components/blog-sidebar.blade.php

<form action="{{ route('blog.buscar') }}" method="GET">
    <input 
        type="text" 
        name="q"
        placeholder="Buscar artículos..."
        class="w-full pl-10 pr-4 py-3 border rounded-lg"
    >
    <button type="submit">Buscar</button>
</form>

💡 El parámetro name="q" es estándar para búsquedas. La URL resultante será /blog/buscar?q=nutricion.

3 Artículos recientes con miniatura

Mostramos los últimos 5 artículos con su imagen pequeña y título.

📍 Archivo: resources/views/components/blog-sidebar.blade.php

@foreach($recientes as $articulo)
    <li class="flex gap-3">
        @if($articulo->imagen_destacada)
            <img src="{{ asset('storage/' . $articulo->imagen_destacada) }}" 
                 class="w-16 h-16 object-cover rounded-lg">
        @else
            <div class="w-16 h-16 bg-gray-100 rounded-lg">
                <!-- Icono por defecto -->
            </div>
        @endif
        <div>
            <a href="{{ $articulo->url }}">{{ $articulo->titulo }}</a>
            <p>{{ $articulo->fecha_publicacion->format('d M, Y') }}</p>
        </div>
    </li>
@endforeach

💡 El @if muestra la imagen si existe, o un placeholder si no tiene.

4 Formulario de Newsletter

Un formulario simple para capturar emails de suscriptores.

📍 Archivo: resources/views/components/blog-sidebar.blade.php

<form action="{{ route('newsletter.suscribir') }}" method="POST">
    @csrf
    <input type="email" name="email" placeholder="tu@email.com" required>
    <button type="submit">Suscribirme</button>
</form>

💡 @csrf es obligatorio en formularios POST de Laravel. Protege contra ataques de falsificación.

📁 ¿Quieres ver el código completo del sidebar? (opcional)

Esto es solo para que veas dónde va tu código. No necesitas memorizar todo esto:

<!-- Sidebar del Blog -->
@php
use App\Models\Articulo;
use App\Models\Categoria;

$recientes = Articulo::publicados()->recientes()->limit(5)->get();
$categorias = Categoria::conArticulosPublicados()->ordenadas()->get();
@endphp

<aside class="space-y-8">
    
    <!-- Buscador -->
    <div class="bg-white rounded-xl shadow-md p-6">
        <h3>Buscar</h3>
        <form action="{{ route('blog.buscar') }}" method="GET">
            <input type="text" name="q" placeholder="Buscar...">
            <button>Buscar</button>
        </form>
    </div>

    <!-- Categorías -->
    <div class="bg-white rounded-xl shadow-md p-6">
        <h3>Categorías</h3>
        <ul>
            @foreach($categorias as $categoria)
                <li>
                    <a href="{{ route('blog.categoria', $categoria->slug) }}">
                        <span style="background: {{ $categoria->color }}"></span>
                        {{ $categoria->nombre }}
                        <span>{{ $categoria->articulos()->publicados()->count() }}</span>
                    </a>
                </li>
            @endforeach
        </ul>
    </div>

    <!-- Recientes -->
    <div class="bg-white rounded-xl shadow-md p-6">
        <h3>Recientes</h3>
        <ul>
            @foreach($recientes as $articulo)
                <li>
                    <img src="{{ asset('storage/' . $articulo->imagen_destacada) }}">
                    <a href="{{ $articulo->url }}">{{ $articulo->titulo }}</a>
                </li>
            @endforeach
        </ul>
    </div>

    <!-- Newsletter -->
    <div class="bg-gradient-to-br from-medico-azul-500 to-medico-azul-600 rounded-xl p-6 text-white">
        <h3>Newsletter</h3>
        <form action="{{ route('newsletter.suscribir') }}" method="POST">
            @csrf
            <input type="email" name="email" placeholder="tu@email.com">
            <button>Suscribirme</button>
        </form>
    </div>
</aside>

🎉 ¿Qué aprendimos hoy?

  • El sidebar consulta datos de la base de datos usando scopes del modelo
  • Las categorías muestran su color identificador y conteo de artículos
  • El buscador usa GET para que la URL sea compartible y bookmarkable
  • Los artículos recientes muestran miniatura o placeholder alternativo
  • El formulario de newsletter usa @csrf para seguridad

¡Tu sidebar está completo! Solo nos falta la paginación para cuando haya muchos artículos. ¡Nos vemos en la última lección! 📑
