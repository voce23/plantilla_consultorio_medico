Módulo 5: Perfil del Médico (About)

En este módulo vamos a crear la sección "Sobre Nosotros" de la clínica. Aprenderás a hacer componentes visuales profesionales: perfiles de médicos, timelines de trayectoria, grids de logos y barras de habilidades animadas.

📚 Lecciones de este módulo:

  • Lección 5.1: Sección About con Blade

    Componente DoctorProfile - Foto con clip-path personalizado, biografía del médico y estadísticas

  • Lección 5.2: Timeline de Trayectoria

    Componente Livewire Timeline - Estudios, especializaciones y cargos con animaciones

  • Lección 5.3: Logos de Instituciones

    Grid de imágenes con efecto grayscale que cambia a color al hacer hover

  • Lección 5.4: Barras de Habilidades

    Alpine.js + CSS para barras animadas que se llenan al entrar en viewport

🎯 Lo que construiremos:

  • Una sección "About" profesional con foto del médico
  • Timeline animado de la trayectoria profesional
  • Grid de logos de instituciones con efectos hover
  • Barras de habilidades que se animan al hacer scroll
  • Resource de Filament para gestionar doctores
