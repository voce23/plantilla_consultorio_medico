Lección 6.5: Confirmación con Modal

Alpine.js modal + resumen de cita

🎯 Piensa en esto: Es como cuando compras online y antes de pagar ves una pantalla de "Revisa tu pedido". Te muestra todo lo que vas a comprar para que confirmes que está correcto. Así funciona nuestro modal de citas.
1 Crear el componente del modal

Usamos Alpine.js para manejar la apertura/cierre y las transiciones.

📍 Archivo: resources/views/components/modal-confirmacion-cita.blade.php

@props([
    'mostrar' => false,
    'cita' => null,
])

<div
    x-data="{ abierto: @js($mostrar) }"
    x-show="abierto"
    class="fixed inset-0 z-50 overflow-y-auto"
    style="display: none;"
    @keydown.escape.window="abierto = false"
>

💡 @js($mostrar) convierte el valor PHP a JavaScript para Alpine.

2 Agregar el overlay y animaciones

El fondo oscuro y las transiciones hacen que el modal se sienta profesional.

📍 Archivo: resources/views/components/modal-confirmacion-cita.blade.php

<!-- Overlay oscuro -->
<div 
    x-show="abierto"
    x-transition:enter="transition ease-out duration-300"
    x-transition:enter-start="opacity-0"
    x-transition:enter-end="opacity-100"
    class="fixed inset-0 bg-gray-900 bg-opacity-50 backdrop-blur-sm"
    @click="abierto = false"
></div>

💡 x-transition define cómo entra y sale el elemento. Aquí fade in/out de 300ms.

📝 ESTE ES EL CÓDIGO QUE VAMOS A APRENDER:

📍 Archivo: resources/views/components/modal-confirmacion-cita.blade.php

<!-- Card del modal -->
<div
    x-show="abierto"
    x-transition:enter="transition ease-out duration-300"
    x-transition:enter-start="opacity-0 transform scale-95 translate-y-4"
    x-transition:enter-end="opacity-100 transform scale-100 translate-y-0"
    class="relative w-full max-w-lg transform overflow-hidden rounded-2xl bg-white shadow-2xl"
    @click.outside="abierto = false"
>
    <!-- Header con gradiente -->
    <div class="bg-gradient-to-r from-medico-azul-500 to-medico-verde-500 px-6 py-4">
        <h3 class="text-xl font-bold text-white">Confirmar Cita</h3>
    </div>
    
    <!-- Contenido -->
    <div class="px-6 py-6">
        @if($cita)
            <p>Paciente: {{ $cita['paciente_nombre'] }}</p>
            <p>Doctor: {{ $cita['doctor_nombre'] }}</p>
            <p>Fecha: {{ $cita['fecha'] }}</p>
            <p>Hora: {{ $cita['hora'] }}</p>
        @endif
    </div>
</div>

🔍 Vamos a entender línea por línea:

x-transition:enter-start="opacity-0 transform scale-95 translate-y-4"
→ Estado inicial de la animación: invisible, más pequeño (95%) y desplazado hacia abajo 4 unidades.
x-transition:enter-end="opacity-100 transform scale-100 translate-y-0"
→ Estado final: visible, tamaño normal y en su posición. La transición crea el efecto de "aparecer y crecer".
@click.outside="abierto = false"
→ Si haces clic fuera del modal, se cierra. Es una forma intuitiva de cancelar.
@keydown.escape.window="abierto = false"
→ También puedes cerrar presionando la tecla ESC. Buena accesibilidad.

📊 Directivas de transición de Alpine.js:

DirectivaCuándo se aplica
x-transition:enterElemento aparece
x-transition:enter-startEstado inicial
x-transition:enter-endEstado final
x-transition:leaveElemento desaparece
x-transition:leave-startEstado inicial de salida
x-transition:leave-endEstado final de salida
📁 Estructura completa del modal 
<div x-data="{ abierto: false }">
    <!-- Botón para abrir -->
    <button @click="abierto = true">Ver Resumen</button>
    
    <!-- Modal -->
    <div x-show="abierto" class="fixed inset-0 z-50">
        <!-- Overlay -->
        <div x-show="abierto" 
             x-transition 
             class="bg-black bg-opacity-50"
             @click="abierto = false"></div>
        
        <!-- Contenido -->
        <div x-show="abierto"
             x-transition:enter="transition ease-out duration-300"
             x-transition:enter-start="opacity-0 scale-95"
             x-transition:enter-end="opacity-100 scale-100"
             class="bg-white rounded-lg p-6">
            <h2>Resumen de Cita</h2>
            <!-- Detalles... -->
            <button @click="abierto = false">Cerrar</button>
        </div>
    </div>
</div>

🎉 ¿Qué aprendimos hoy?

  • x-transition crea animaciones suaves de entrada/salida
  • @click.outside cierra el modal al hacer clic fuera
  • @keydown.escape permite cerrar con tecla ESC
  • transform scale-95 hace el efecto de "aparecer creciendo"
  • backdrop-blur aplica desenfoque al fondo
