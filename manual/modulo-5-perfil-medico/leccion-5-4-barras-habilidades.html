Lección 5.4: Barras de Habilidades

Alpine.js + CSS para barras animadas al entrar en viewport

🎯 Piensa en esto: ¿Recuerdas las barras de carga de los videojuegos? Empiezan vacías y se van llenando poco a poco. Las barras de habilidades son iguales: cuando alguien hace scroll y las ve, cobran vida y se llenan hasta el nivel de experiencia del médico. Es una forma visual y satisfactoria de mostrar competencia.
1 Crear el componente de habilidades

Vamos a hacer barras que se animan cuando aparecen en pantalla. Usaremos Alpine.js para detectar cuándo son visibles.

📍 Archivo: resources/views/components/barras-habilidades.blade.php

@props([
    'habilidades' => [],
    'titulo' => 'Áreas de Especialización'
])

💡 Cada habilidad tiene: nombre, porcentaje y color (medico-azul o medico-verde).

2 Definir la función Alpine.js

Alpine.js usa funciones JavaScript para manejar el estado. Crearemos una función que controle cuándo iniciar la animación.

📍 Archivo: resources/views/components/barras-habilidades.blade.php

<script>
function skillsAnimation() {
    return {
        startAnimation: false,
    }
}
</script>

💡 Esta función devuelve un objeto con una variable startAnimation que será false al inicio.

3 Conectar Alpine.js con el componente

Usamos x-data para inicializar Alpine y x-intersect para detectar cuando el componente es visible.

📍 Archivo: resources/views/components/barras-habilidades.blade.php

<section 
    class="py-20 bg-white"
    x-data="skillsAnimation()"
    x-intersect="startAnimation = true"
>
    <!-- Contenido -->
</section>

💡 x-intersect detecta cuando el elemento entra en el viewport (lo que ves en pantalla).

📝 ESTE ES EL CÓDIGO QUE VAMOS A APRENDER:

📍 Archivo: resources/views/components/barras-habilidades.blade.php

<!-- Cada barra de habilidad -->
<div class="group"
     x-data="{ visible: false, width: 0 }"
     x-intersect="visible = true"
     x-init="
         $watch('startAnimation', (value) => {
             if(value && visible) {
                 setTimeout(() => {
                     width = {{ $habilidad['porcentaje'] }};
                 }, {{ $index * 150 }});
             }
         })
     ">
    
    <!-- Título y porcentaje -->
    <div class="flex justify-between items-center mb-2">
        <h3>{{ $habilidad['nombre'] }}</h3>
        <span x-text="width + '%'">0%</span>
    </div>
    
    <!-- Barra de progreso -->
    <div class="relative h-4 bg-gray-200 rounded-full overflow-hidden">
        <div class="absolute inset-y-0 left-0 rounded-full transition-all duration-1000 ease-out 
                    bg-gradient-to-r from-medico-azul-400 to-medico-azul-600"
             :style="'width: ' + width + '%'">
        </div>
    </div>
</div>

🔍 Vamos a entender línea por línea (como si fuera la primera vez):

x-data="{ visible: false, width: 0 }"
→ Cada barra tiene su propio estado: visible (¿está en pantalla?) y width (¿cuánto debe medir la barra?).
x-intersect="visible = true"
→ Cuando la barra entra en el viewport, cambia visible a true. Es como decir "¡Ya me vieron!".
$watch('startAnimation', (value) => {...})
→ $watch vigila una variable. Cuando startAnimation cambia, ejecuta el código dentro. Es un "observador".
setTimeout(() => {...}, {{ $index * 150 }})
→ Espera un tiempo antes de ejecutar. $index * 150 hace que cada barra se anime con 150ms de diferencia. La primera en 0ms, la segunda en 150ms, etc.
:style="'width: ' + width + '%'"
→ Los dos puntos (:) indican un "binding" dinámico. El estilo cambia cuando width cambia. Si width es 90, el estilo será "width: 90%".
x-text="width + '%'"
→ Actualiza el texto del elemento con el valor de width. Muestra "0%", luego "50%", hasta llegar al porcentaje final.

📊 Directivas de Alpine.js que usamos:

Directiva Qué hace Ejemplo
x-data Inicializa variables locales x-data="{ width: 0 }"
x-intersect Detecta entrada en viewport x-intersect="visible = true"
x-init Código al inicializar x-init="$watch(...)"
$watch Vigila cambios en variable $watch('startAnimation', fn)
:style Estilo dinámico :style="'width: ' + w + '%'"
x-text Texto dinámico x-text="width + '%'"

👆 El símbolo ":" es abreviatura de x-bind: para valores dinámicos.

📁 ¿Quieres ver el código completo del archivo? (opcional)

Este es el componente completo de barras de habilidades:

<!-- Barras de Habilidades Animadas -->
<!-- 📁 Archivo: resources/views/components/barras-habilidades.blade.php -->

@props([
    'habilidades' => [],
    'titulo' => 'Áreas de Especialización'
])

<section 
    class="py-20 bg-white"
    x-data="skillsAnimation()"
    x-intersect="startAnimation = true"
>
    <div class="px-4 mx-auto max-w-4xl sm:px-6 lg:px-8">
        <div class="mb-12 text-center">
            <h2 class="text-3xl font-bold text-gray-900 sm:text-4xl font-serif">
                {{ $titulo }}
            </h2>
            <p class="mt-4 text-lg text-gray-600">
                Experiencia y dominio en diversas áreas médicas
            </p>
        </div>

        @php
            $habilidadesDefault = [
                ['nombre' => 'Cardiología Clínica', 'porcentaje' => 95, 'color' => 'medico-azul'],
                ['nombre' => 'Ecocardiografía', 'porcentaje' => 90, 'color' => 'medico-verde'],
                ['nombre' => 'Medicina Preventiva', 'porcentaje' => 85, 'color' => 'medico-azul'],
                ['nombre' => 'Diagnóstico Clínico', 'porcentaje' => 92, 'color' => 'medico-verde'],
                ['nombre' => 'Atención al Paciente', 'porcentaje' => 98, 'color' => 'medico-azul'],
            ];
            $habilidades = !empty($habilidades) ? $habilidades : $habilidadesDefault;
        @endphp

        <div class="space-y-8">
            @foreach($habilidades as $index => $habilidad)
                <div class="group"
                     x-data="{ visible: false, width: 0 }"
                     x-intersect="visible = true"
                     x-init="
                         $watch('startAnimation', (value) => {
                             if(value && visible) {
                                 setTimeout(() => {
                                     width = {{ $habilidad['porcentaje'] }};
                                 }, {{ $index * 150 }});
                             }
                         })
                     ">
                    
                    <div class="flex justify-between items-center mb-2">
                        <h3 class="text-lg font-medium text-gray-900">
                            {{ $habilidad['nombre'] }}
                        </h3>
                        <span class="text-lg font-bold text-{{ $habilidad['color'] }}-600"
                              x-text="width + '%'">0%</span>
                    </div>

                    <div class="relative h-4 bg-gray-200 rounded-full overflow-hidden">
                        <div class="absolute inset-y-0 left-0 rounded-full transition-all duration-1000 ease-out 
                                    bg-gradient-to-r from-{{ $habilidad['color'] }}-400 to-{{ $habilidad['color'] }}-600"
                             :style="'width: ' + width + '%'">
                            <div class="absolute inset-0 bg-gradient-to-r from-transparent via-white to-transparent opacity-30 animate-pulse"></div>
                        </div>
                    </div>
                </div>
            @endforeach
        </div>

        <!-- Resumen -->
        <div class="mt-12 p-6 bg-gradient-to-r from-medico-azul-50 to-medico-verde-50 rounded-2xl">
            <div class="flex items-center justify-center space-x-8">
                <div class="text-center">
                    <div class="text-3xl font-bold text-medico-azul-600">{{ count($habilidades) }}</div>
                    <div class="text-sm text-gray-600">Especialidades</div>
                </div>
                <div class="w-px h-12 bg-gray-300"></div>
                <div class="text-center">
                    <div class="text-3xl font-bold text-medico-verde-600">
                        {{ round(collect($habilidades)->avg('porcentaje')) }}%
                    </div>
                    <div class="text-sm text-gray-600">Promedio</div>
                </div>
            </div>
        </div>
    </div>
</section>

<script>
function skillsAnimation() {
    return {
        startAnimation: false,
    }
}
</script>

🎉 ¿Qué aprendimos hoy?

  • x-data inicializa variables en Alpine.js
  • x-intersect detecta cuando un elemento entra en el viewport
  • $watch() vigila cambios en una variable
  • :style permite estilos dinámicos basados en variables
  • setTimeout() crea retrasos para animaciones escalonadas
  • La combinación de Alpine.js + CSS transitions crea animaciones fluidas
