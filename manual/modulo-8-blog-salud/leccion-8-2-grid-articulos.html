Lección 8.2: Grid de Artículos

Filament Table con ImageColumn y badges de categoría

🎯 Piensa en esto: Imagina una revista donde ves portadas de artículos con fotos atractivas, etiquetas de colores indicando la sección, y puedes filtrar para ver solo "Nutrición" o "Cardiología". Eso es lo que vamos a crear en el panel de administración.
1 Mostrar la imagen destacada

La ImageColumn muestra miniaturas de las fotos directamente en la tabla.

📍 Archivo: app/Filament/Resources/Articulos/Tables/ArticulosTable.php

ImageColumn::make('imagen_destacada')
    ->label('Imagen')
    ->square()
    ->defaultImageUrl(fn () => 'https://placehold.co/100x100?text=Sin+Imagen')
    ->size(60)

💡 square() hace que las imágenes sean cuadradas, y size(60) las hace pequeñas (60x60 píxeles).

📝 ESTE ES EL CÓDIGO QUE VAMOS A APRENDER:

📍 Archivo: app/Filament/Resources/Articulos/Tables/ArticulosTable.php

TextColumn::make('categoria.nombre')
    ->label('Categoría')
    ->badge()
    ->color(fn ($record) => $record->categoria?->color ?? 'gray')
    ->sortable()

🔍 Vamos a entender línea por línea (como si fuera la primera vez):

TextColumn::make('categoria.nombre')
→ Accedemos al nombre de la categoría a través de la relación. El punto (.) navega entre modelos relacionados.
->badge()
→ Transforma el texto en una etiqueta tipo "pill" o pastilla, como las etiquetas de GitHub.
->color(fn ($record) => $record->categoria?->color ?? 'gray')
→ Aquí está la magia: el color del badge viene del campo color de la categoría. El operador ?? significa "si no existe, usa gris".
?-> (null safe operator)
→ Si la categoría es null (no tiene categoría), no da error, simplemente devuelve null. Es como decir "si existe, toma el color, si no, no pasa nada".
2 Mostrar el contador de visitas

Es útil ver qué artículos son más populares directamente en la tabla.

📍 Archivo: app/Filament/Resources/Articulos/Tables/ArticulosTable.php

TextColumn::make('visitas')
    ->label('Visitas')
    ->numeric()
    ->sortable()
    ->icon('heroicon-o-eye')
    ->color('success')

💡 El icono de ojo y el color verde hacen que las visitas destaquen visualmente.

3 Filtro por categoría

Permite filtrar rápidamente para ver solo los artículos de una categoría específica.

📍 Archivo: app/Filament/Resources/Articulos/Tables/ArticulosTable.php

SelectFilter::make('categoria')
    ->label('Categoría')
    ->relationship('categoria', 'nombre')
    ->searchable()
    ->preload()

💡 preload() carga todas las categorías al inicio para que el filtro sea instantáneo.

4 Toggle para publicar/despublicar

Un interruptor directo en la tabla para cambiar el estado sin entrar al formulario.

📍 Archivo: app/Filament/Resources/Articulos/Tables/ArticulosTable.php

ToggleColumn::make('publicado')
    ->label('Publicado')
    ->sortable()

💡 Con un solo clic puedes publicar o despublicar un artículo. Muy práctico.

📁 ¿Quieres ver el código completo de la tabla? (opcional)

Esto es solo para que veas dónde va tu código. No necesitas memorizar todo esto:

<?php

namespace App\Filament\Resources\Articulos\Tables;

use Filament\Tables\Columns\ImageColumn;
use Filament\Tables\Columns\TextColumn;
use Filament\Tables\Columns\ToggleColumn;
use Filament\Tables\Filters\SelectFilter;
use Filament\Tables\Table;

class ArticulosTable
{
    public static function configure(Table $table): Table
    {
        return $table
            ->columns([
                ImageColumn::make('imagen_destacada')
                    ->label('Imagen')
                    ->square()
                    ->size(60),

                TextColumn::make('titulo')
                    ->label('Título')
                    ->searchable()
                    ->wrap()
                    ->weight('bold'),

                TextColumn::make('categoria.nombre')
                    ->label('Categoría')
                    ->badge()
                    ->color(fn ($record) => $record->categoria?->color ?? 'gray'),

                TextColumn::make('visitas')
                    ->label('Visitas')
                    ->numeric()
                    ->icon('heroicon-o-eye')
                    ->color('success'),

                ToggleColumn::make('publicado')
                    ->label('Publicado'),
            ])
            ->filters([
                SelectFilter::make('categoria')
                    ->relationship('categoria', 'nombre')
                    ->searchable()
                    ->preload(),
            ])
            ->defaultSort('fecha_publicacion', 'desc');
    }
}

🎉 ¿Qué aprendimos hoy?

  • ImageColumn muestra miniaturas de las imágenes en la tabla
  • Los badge() con colores dinámicos hacen que las categorías sean visuales
  • El operador ?-> (null safe) evita errores cuando no hay relación
  • ToggleColumn permite cambiar estados con un solo clic
  • SelectFilter con relationship filtra por categorías fácilmente
