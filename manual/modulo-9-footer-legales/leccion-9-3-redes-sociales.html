Lección 9.3: Iconos de Redes Sociales

Componente Blade con SVG inline + hover colors

🎯 Piensa en esto: Imagina que tienes tarjetas de presentación de diferentes colores, una para cada red social. Cuando alguien te pide tu contacto, le das la tarjeta correspondiente. En tu sitio web, estos iconos son como esas tarjetas: cada uno lleva a una red social diferente y cambia de color cuando el usuario pasa el mouse.
1 Crear el componente con props configurables

Cada red social es opcional. Si no pasas una URL, el icono no aparece.

📍 Archivo: resources/views/components/redes-sociales.blade.php

@props([
    'class' => '',
    'facebook' => '#',
    'twitter' => '#',
    'instagram' => '#',
    'linkedin' => '#',
    'youtube' => null,    // Opcional
    'whatsapp' => null,   // Opcional
    'tamaño' => 'md',     // sm, md, lg
])

💡 null significa que el icono no se muestra a menos que pases una URL.

📝 ESTE ES EL CÓDIGO QUE VAMOS A APRENDER:

📍 Archivo: resources/views/components/redes-sociales.blade.php

<!-- Facebook con hover azul -->
<a 
    href="{{ $facebook }}"
    target="_blank"
    rel="noopener noreferrer"
    class="w-10 h-10 bg-white/10 rounded-full flex items-center justify-center 
           text-white hover:bg-[#1877F2] transition-all duration-300"
    aria-label="Facebook"
>
    <svg class="w-5 h-5" fill="currentColor" viewBox="0 0 24 24">
        <path d="M24 12.073c0-6.627-5.373-12-12-12s-12 5.373-12 12..."/>
    </svg>
</a>

🔍 Vamos a entender línea por línea (como si fuera la primera vez):

target="_blank"
→ Abre el link en una nueva pestaña para que el usuario no abandone tu sitio.
rel="noopener noreferrer"
→ Medida de seguridad importante cuando usas target="_blank". Evita que la página externa acceda a tu ventana.
bg-white/10
→ Fondo blanco con 10% de opacidad. Se ve bien sobre fondos oscuros y el hover resalta más.
hover:bg-[#1877F2]
→ El color azul oficial de Facebook (usando su código HEX exacto). Cada red tiene su color.
transition-all duration-300
→ La animación dura 300ms y suaviza el cambio de color. Sin esto, el cambio sería brusco.
aria-label="Facebook"
→ Ayuda a los lectores de pantalla a describir el icono para personas con discapacidad visual.
2 SVG inline vs imágenes externas

Usamos SVG inline en lugar de imágenes PNG/JPG por varias razones:

  • Escalables: Se ven nítidos en cualquier tamaño
  • Coloreables: Podemos cambiar el color con CSS (fill="currentColor")
  • Ligeros: Un SVG pesa menos que una imagen
  • Sin requests extra: No necesitan descargarse del servidor
3 Colores de hover por red social

Cada red tiene su color oficial. Aquí están los códigos HEX:

Facebook:  #1877F2 (azul)
Twitter:   #1DA1F2 (azul claro)
Instagram: gradiente #833AB4 → #FD1D1D → #F77737
LinkedIn:  #0A66C2 (azul corporativo)
YouTube:   #FF0000 (rojo)
WhatsApp:  #25D366 (verde)

💡 Instagram usa hover:bg-gradient-to-br para crear el degradé característico.

4 Tamaños disponibles

El componente permite 3 tamaños mediante un array asociativo:

@php
$tamaños = [
    'sm' => 'w-8 h-8',   // 32px - Para headers compactos
    'md' => 'w-10 h-10', // 40px - Tamaño por defecto
    'lg' => 'w-12 h-12', // 48px - Para destacar
];
$iconoTamaño = $tamaños[$tamaño] ?? $tamaños['md'];
@endphp

💡 El operador ?? (null coalescing) usa 'md' si el tamaño no existe.

5 WhatsApp con formato especial

WhatsApp necesita un formato especial para abrir la app directamente:

@if($whatsapp)
    <a href="https://wa.me/{{ preg_replace('/[^0-9]/', '', $whatsapp) }}">
        {{-- Icono de WhatsApp --}}
    </a>
@endif

💡 preg_replace('/[^0-9]/', '', $whatsapp) elimina espacios, guiones y paréntesis, dejando solo números.

6 Usar el componente

Es muy sencillo incluirlo en cualquier vista:

📍 Archivo: Cualquier vista Blade

<!-- En el footer con todas las redes -->
<x-redes-sociales 
    facebook="https://facebook.com/clinica"
    instagram="https://instagram.com/clinica"
    linkedin="https://linkedin.com/company/clinica"
    whatsapp="+52 123 456 7890"
    tamaño="md"
/>

<!-- En el header solo con las principales -->
<x-redes-sociales 
    facebook="https://facebook.com/clinica"
    twitter="https://twitter.com/clinica"
    tamaño="sm"
    class="ml-auto"
/>

💡 La prop class permite añadir clases adicionales como márgenes o alineación.

🎉 ¿Qué aprendimos hoy?

  • SVG inline es mejor que imágenes porque es escalable y coloreable
  • currentColor permite que el SVG herede el color del texto
  • Cada red social tiene su color oficial HEX para el hover
  • rel="noopener noreferrer" es obligatorio con target="_blank"
  • WhatsApp usa wa.me para abrir directamente la app
  • Las props con null permiten iconos opcionales
