Lección 7.4: Widget de Google Reviews

Componente Blade con estrellas y promedio estático

🎯 Piensa en esto: Imagina que entras a un restaurante y ves un certificado enmarcado en la pared que dice "4.8 estrellas - 500 reseñas en Google". Inmediatamente sientes más confianza para pedir tu comida, ¿verdad? Ese certificado es social proof: la evidencia de que otros ya confiaron y quedaron satisfechos. Este widget es exactamente eso, pero para tu clínica.
1 Crear el componente Blade

Vamos a crear un componente que muestre el promedio de reseñas, las estrellas visuales y un botón para que los visitantes dejen su propia reseña.

📍 Archivo: resources/views/components/google-reviews-widget.blade.php

# Creamos el archivo directamente

💡 Este es un componente "presentacional": solo muestra datos, no tiene lógica compleja.

2 Definir las props con valores por defecto

Queremos que sea fácil de usar, con valores realistas por defecto que el usuario pueda cambiar si quiere.

📍 Archivo: resources/views/components/google-reviews-widget.blade.php

@props([
    'promedio' => 4.8,
    'totalResenas' => 127,
    'enlaceGoogle' => '#',
])

💡 El # es un placeholder. En producción, pondrías el enlace real a tu perfil de Google Business.

📝 ESTE ES EL CÓDIGO QUE VAMOS A APRENDER:

📍 Archivo: resources/views/components/google-reviews-widget.blade.php

@for($i = 1; $i <= 5; $i++)
    @if($i <= floor($promedio))
        <!-- Estrella completa -->
        <svg class="w-6 h-6 text-yellow-400" fill="currentColor">
            <path d="M9.049 2.927..."/>
        </svg>
    @elseif($i == ceil($promedio) && $promedio != floor($promedio))
        <!-- Media estrella -->
        <svg class="w-6 h-6">
            <defs>
                <linearGradient id="half-{{ $i }}">
                    <stop offset="50%" stop-color="#FBBF24"/>
                    <stop offset="50%" stop-color="#D1D5DB"/>
                </linearGradient>
            </defs>
            <path fill="url(#half-{{ $i }})" d="M9.049 2.927..."/>
        </svg>
    @else
        <!-- Estrella vacía -->
        <svg class="w-6 h-6 text-gray-300" fill="currentColor">
            <path d="M9.049 2.927..."/>
        </svg>
    @endif
@endfor

🔍 Vamos a entender línea por línea (como si fuera la primera vez):

@for($i = 1; $i <= 5; $i++)
→ Un bucle que se repite 5 veces (una por cada estrella). La variable $i va del 1 al 5.
@if($i <= floor($promedio))
→ Si el número de estrella es menor o igual al promedio redondeado hacia abajo, pintamos una estrella completa dorada. Por ejemplo, si el promedio es 4.8, las estrellas 1, 2, 3 y 4 son completas.
@elseif($i == ceil($promedio) && $promedio != floor($promedio))
→ Aquí está el truco de la media estrella: si estamos en la estrella que coincide con el promedio redondeado hacia arriba, Y el promedio no es un número entero, mostramos media estrella. Para 4.8, ceil(4.8) es 5, así que la estrella 5 es media.
<linearGradient id="half-{{ $i }}">
→ Creamos un degradado que va de amarillo (50%) a gris (50%). Esto simula una estrella medio pintada. Cada estrella necesita su propio ID único.
@else
→ Si no es estrella completa ni media, es una estrella vacía gris. Estas son las estrellas "sobrantes" si el promedio es bajo.
3 La barra de distribución de estrellas

Google muestra barras que indican cuántas reseñas de cada tipo (5 estrellas, 4 estrellas, etc.) tiene el negocio.

📍 Archivo: resources/views/components/google-reviews-widget.blade.php

@foreach([5, 4, 3, 2, 1] as $estrella)
    @php
        $porcentaje = match($estrella) {
            5 => 75, 4 => 15, 3 => 5, 2 => 3, 1 => 2,
            default => 0
        };
    @endphp
    <div class="flex items-center gap-3">
        <span>{{ $estrella }} estrellas</span>
        <div class="flex-1 h-3 bg-gray-100 rounded-full">
            <div class="h-full bg-yellow-400 rounded-full" 
                 style="width: {{ $porcentaje }}%">
            </div>
        </div>
        <span>{{ $porcentaje }}%</span>
    </div>
@endforeach

💡 El match de PHP 8 es como un switch más limpio. Asigna un porcentaje según la cantidad de estrellas.

5 Usar el widget en tu página

Es tan fácil como usar cualquier etiqueta HTML, pero con datos personalizados.

📍 Archivo: Cualquier vista Blade

<!-- Con valores por defecto -->
<x-google-reviews-widget />

<!-- Con valores personalizados -->
<x-google-reviews-widget 
    promedio="4.9"
    :total-resenas="156"
    enlace-google="https://g.page/r/..."
/>

💡 Fíjate que :total-resenas tiene dos puntos porque es un número, no un string.

🎉 ¿Qué aprendimos hoy?

  • El "social proof" (prueba social) aumenta la confianza de los visitantes
  • Las estrellas se renderizan con lógica condicional: completas, medias o vacías según el promedio
  • linearGradient en SVG permite crear la ilusión de media estrella pintada
  • Las barras de distribución dan credibilidad mostrando el desglose de calificaciones
  • Los componentes Blade con props hacen que sea fácil reutilizar y personalizar el widget
