Módulo 6: Sistema de Citas con Filament Forms

En este módulo construiremos un sistema completo de reserva de citas médicas. Desde el modelo de datos hasta la integración con WhatsApp para confirmaciones instantáneas.

📚 Lecciones de este módulo:

  • Lección 6.1: Modelo y Migración de Citas

    Crear el modelo Cita con sus relaciones y la migración de base de datos

  • Lección 6.2: Formulario de Citas en Filament

    Crear el Resource de Citas con formulario completo en el panel de administración

  • Lección 6.3: Selector de Fecha con Calendar

    Filament DatePicker con validación de disponibilidad de doctores

  • Lección 6.4: Grid de Horarios Disponibles

    Componente Livewire HorarioSelector con toggle de horarios ocupados/libres

  • Lección 6.5: Confirmación con Modal

    Alpine.js modal con resumen de cita antes de confirmar

  • Lección 6.6: Integración WhatsApp

    Botón dinámico con https://wa.me/ que incluye datos del formulario

🎯 Lo que construiremos:

  • Modelo Cita con relaciones a Paciente, Doctor y Especialidad
  • Resource de Filament para gestionar citas desde el admin
  • Selector de fechas con validación de disponibilidad
  • Grid de horarios con indicadores visuales (libre/ocupado)
  • Modal de confirmación con resumen de la cita
  • Integración con WhatsApp para enviar confirmaciones
