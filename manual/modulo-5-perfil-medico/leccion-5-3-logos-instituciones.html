Lección 5.3: Logos de Instituciones

Grid de imágenes con grayscale + color en hover

🎯 Piensa en esto: ¿Has visto esos muros de empresas donde cada logo está en blanco y negro, pero cuando pasas el mouse se ilumina en color? Es como una galería de arte que cobra vida. Ese efecto "grayscale a color" transmite profesionalismo y hace que la interacción sea satisfactoria.
1 Crear el componente de logos

Vamos a hacer un componente que muestre una cuadrícula de logos. Si no hay logos reales, mostraremos placeholders con las iniciales.

📍 Archivo: resources/views/components/logos-instituciones.blade.php

@props([
    'instituciones' => [],
    'titulo' => 'Instituciones donde hemos colaborado'
])

💡 Este componente acepta una lista de instituciones y un título personalizable.

2 Definir datos de ejemplo con @php

Cuando no tenemos logos reales, necesitamos datos de ejemplo. Blade permite ejecutar PHP directamente con @php.

📍 Archivo: resources/views/components/logos-instituciones.blade.php

@php
    $institucionesDefault = [
        ['nombre' => 'Hospital Nacional', 'logo' => null],
        ['nombre' => 'Clínica Internacional', 'logo' => null],
        ['nombre' => 'Universidad de Medicina', 'logo' => null],
    ];
    $instituciones = !empty($instituciones) ? $instituciones : $institucionesDefault;
@endphp

💡 El operador ternario ? : es un if/else en una línea: "¿hay instituciones? úsalas, si no, usa las default".

3 Crear el grid de logos

Usaremos CSS Grid para organizar los logos en filas y columnas automáticas.

📍 Archivo: resources/views/components/logos-instituciones.blade.php

<div class="grid grid-cols-2 gap-8 md:grid-cols-4 lg:grid-cols-6">
    @foreach($instituciones as $index => $institucion)
        <div class="group relative flex items-center justify-center p-6 bg-white rounded-xl">
            <!-- Logo o placeholder -->
        </div>
    @endforeach
</div>

💡 grid-cols-2 = 2 columnas en móvil, md:grid-cols-4 = 4 columnas en tablet, lg:grid-cols-6 = 6 columnas en desktop.

📝 ESTE ES EL CÓDIGO QUE VAMOS A APRENDER:

📍 Archivo: resources/views/components/logos-instituciones.blade.php

<!-- Logo con efecto grayscale -->
@if(isset($institucion['logo']) && $institucion['logo'])
    <img src="{{ asset('storage/' . $institucion['logo']) }}" 
         alt="{{ $institucion['nombre'] }}"
         class="h-12 w-auto object-contain 
                filter grayscale group-hover:grayscale-0 
                transition-all duration-500">
@else
    <!-- Placeholder con iniciales -->
    <div class="w-16 h-16 rounded-full 
                bg-gradient-to-br from-gray-200 to-gray-300 
                group-hover:from-medico-azul-100 group-hover:to-medico-verde-100 
                flex items-center justify-center transition-all duration-500">
        <span class="text-xl font-bold text-gray-500 
                     group-hover:text-medico-azul-600 transition-colors">
            {{ collect(explode(' ', $institucion['nombre']))
                ->take(2)
                ->map(fn($w) => Str::substr($w, 0, 1))
                ->join('') }}
        </span>
    </div>
@endif

🔍 Vamos a entender línea por línea (como si fuera la primera vez):

filter grayscale
→ Clase de Tailwind que convierte la imagen a blanco y negro. Es como poner un filtro de Instagram "Noir".
group-hover:grayscale-0
→ Cuando el contenedor padre (con clase group) recibe hover, quita el filtro grayscale. El logo vuelve a color.
transition-all duration-500
→ Hace que el cambio sea suave durante 500 milisegundos (medio segundo). Sin esto, el cambio sería instantáneo y brusco.
explode(' ', $institucion['nombre'])
→ Función PHP que divide un texto en partes usando un separador. "Hospital Nacional" → ["Hospital", "Nacional"].
->map(fn($w) => Str::substr($w, 0, 1))
→ Toma cada palabra y extrae solo la primera letra. ["Hospital", "Nacional"] → ["H", "N"]. La función fn() es una "arrow function" de PHP.

📊 Clases de Tailwind para el efecto hover:

Clase Estado inicial Estado hover
grayscale Blanco y negro -
group-hover:grayscale-0 - Color normal
transition-all duration-500 Animación suave de 0.5s
group-hover:shadow-md Sin sombra Sombra suave

👆 El truco está en poner group en el padre y group-hover: en los hijos.

📁 ¿Quieres ver el código completo del archivo? (opcional)

Este es el componente completo de logos:

<!-- Logos de Instituciones -->
<!-- 📁 Archivo: resources/views/components/logos-instituciones.blade.php -->

@props([
    'instituciones' => [],
    'titulo' => 'Instituciones donde hemos colaborado'
])

<section class="py-16 bg-clinico-gris">
    <div class="px-4 mx-auto max-w-7xl sm:px-6 lg:px-8">
        <div class="mb-12 text-center">
            <h2 class="text-2xl font-bold text-gray-900 sm:text-3xl font-serif">
                {{ $titulo }}
            </h2>
            <p class="mt-4 text-gray-600">
                Reconocidos por las mejores instituciones de salud
            </p>
        </div>

        <div class="grid grid-cols-2 gap-8 md:grid-cols-4 lg:grid-cols-6" 
             x-data="{ hoveredIndex: null }">
            @php
                $institucionesDefault = [
                    ['nombre' => 'Hospital Nacional', 'logo' => null],
                    ['nombre' => 'Clínica Internacional', 'logo' => null],
                    ['nombre' => 'Universidad de Medicina', 'logo' => null],
                    ['nombre' => 'Centro de Salud Premium', 'logo' => null],
                    ['nombre' => 'Instituto Cardiológico', 'logo' => null],
                    ['nombre' => 'Hospital San Juan', 'logo' => null],
                ];
                $instituciones = !empty($instituciones) ? $instituciones : $institucionesDefault;
            @endphp

            @foreach($instituciones as $index => $institucion)
                <div class="group relative flex items-center justify-center p-6 bg-white rounded-xl shadow-sm hover:shadow-md transition-all duration-300"
                     x-on:mouseenter="hoveredIndex = {{ $index }}"
                     x-on:mouseleave="hoveredIndex = null">
                    
                    @if(isset($institucion['logo']) && $institucion['logo'])
                        <img src="{{ asset('storage/' . $institucion['logo']) }}" 
                             alt="{{ $institucion['nombre'] }}"
                             class="h-12 w-auto object-contain filter grayscale group-hover:grayscale-0 transition-all duration-500">
                    @else
                        <div class="flex flex-col items-center">
                            <div class="w-16 h-16 rounded-full bg-gradient-to-br from-gray-200 to-gray-300 group-hover:from-medico-azul-100 group-hover:to-medico-verde-100 flex items-center justify-center transition-all duration-500">
                                <span class="text-xl font-bold text-gray-500 group-hover:text-medico-azul-600 transition-colors duration-300">
                                    {{ collect(explode(' ', $institucion['nombre']))->take(2)->map(fn($w) => Str::substr($w, 0, 1))->join('') }}
                                </span>
                            </div>
                            <span class="mt-2 text-xs text-gray-500 text-center opacity-0 group-hover:opacity-100 transition-opacity duration-300">
                                {{ Str::limit($institucion['nombre'], 20) }}
                            </span>
                        </div>
                    @endif

                    <!-- Tooltip -->
                    <div class="absolute -bottom-2 left-1/2 transform -translate-x-1/2 translate-y-full opacity-0 group-hover:opacity-100 transition-opacity duration-300 pointer-events-none z-10">
                        <div class="px-3 py-2 text-sm text-white bg-gray-900 rounded-lg shadow-lg whitespace-nowrap">
                            {{ $institucion['nombre'] }}
                        </div>
                    </div>
                </div>
            @endforeach
        </div>
    </div>
</section>

🎉 ¿Qué aprendimos hoy?

  • filter grayscale convierte imágenes a blanco y negro
  • group-hover: aplica estilos cuando el padre tiene hover
  • transition-all duration-500 suaviza los cambios
  • explode() divide strings, map() transforma cada elemento
  • Las arrow functions de PHP usan fn($var) => expresion
