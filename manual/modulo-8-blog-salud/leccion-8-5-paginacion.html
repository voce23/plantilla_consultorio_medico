Lección 8.5: Paginación Estilizada

Tailwind pagination personalizada con colores médicos

🎯 Piensa en esto: Imagina un libro de 500 páginas. No puedes mostrar todas las páginas a la vez, así que pones números en la parte inferior para que el lector pueda saltar a la página que quiera. La paginación web funciona igual: divide contenido largo en páginas manejables.
1 Crear la vista de paginación personalizada

Laravel usa vistas específicas para la paginación. Vamos a crear una con nuestros colores médicos.

📍 Terminal:

# Crear la carpeta y archivo
mkdir -p resources/views/vendor/pagination

💡 Laravel busca primero en resources/views/vendor antes que en sus vistas internas.

📝 ESTE ES EL CÓDIGO QUE VAMOS A APRENDER:

📍 Archivo: resources/views/vendor/pagination/tailwind-medico.blade.php

@if ($page == $paginator->currentPage())
    <span class="bg-medico-azul-500 text-white border-medico-azul-500">
        {{ $page }}
    </span>
@else
    <a href="{{ $url }}" class="hover:bg-medico-azul-50 hover:text-medico-azul-600">
        {{ $page }}
    </a>
@endif

🔍 Vamos a entender línea por línea (como si fuera la primera vez):

@if ($page == $paginator->currentPage())
→ Verificamos si este número de página es la página actual. Si es así, lo mostramos destacado.
class="bg-medico-azul-500 text-white"
→ La página actual tiene fondo azul médico y texto blanco para destacar visualmente.
class="hover:bg-medico-azul-50 hover:text-medico-azul-600"
→ Las páginas no activas cambian de color suavemente cuando pasas el mouse por encima.
$paginator->hasMorePages()
→ Una función que devuelve true si hay más páginas después de la actual. Usada para mostrar/ocultar el botón "Siguiente".
2 La estructura completa de la paginación

La paginación tiene tres partes: botón anterior, números de página, y botón siguiente.

📍 Archivo: resources/views/vendor/pagination/tailwind-medico.blade.php

<nav role="navigation">
    {{-- Versión Mobile --}}
    <div class="flex justify-between sm:hidden">
        <a href="{{ $paginator->previousPageUrl() }}">← Anterior</a>
        <a href="{{ $paginator->nextPageUrl() }}">Siguiente →</a>
    </div>

    {{-- Versión Desktop --}}
    <div class="hidden sm:flex">
        {{-- Botón Anterior --}}
        @if ($paginator->onFirstPage())
            <span>←</span>
        @else
            <a href="{{ $paginator->previousPageUrl() }}">←</a>
        @endif

        {{-- Números de página --}}
        @foreach ($elements as $element)
            {{-- ... números ... --}}
        @endforeach

        {{-- Botón Siguiente --}}
        @if ($paginator->hasMorePages())
            <a href="{{ $paginator->nextPageUrl() }}">→</a>
        @else
            <span>→</span>
        @endif
    </div>
</nav>

💡 Tenemos dos versiones: una simple para móviles y otra completa para desktop.

3 Usar la paginación personalizada

En tu controlador o vista, le indicas a Laravel que use tu vista personalizada.

📍 Archivo: Controlador o Vista Blade

// En el controlador
$articulos = Articulo::publicados()
    ->paginate(9)
    ->withQueryString();

// En la vista Blade
{{ $articulos->links('vendor.pagination.tailwind-medico') }}

💡 withQueryString() mantiene los parámetros de búsqueda al cambiar de página.

4 Información de resultados

Muestra "Mostrando X a Y de Z artículos" para que el usuario sepa dónde está.

📍 Archivo: resources/views/vendor/pagination/tailwind-medico.blade.php

<p>
    Mostrando 
    <span>{{ $paginator->firstItem() }}</span>
    a
    <span>{{ $paginator->lastItem() }}</span>
    de
    <span>{{ $paginator->total() }}</span>
    artículos
</p>

💡 Estos métodos del paginador calculan automáticamente los números correctos.

📁 ¿Quieres ver el código completo de la paginación? (opcional)

Esto es solo para que veas dónde va tu código. No necesitas memorizar todo esto:

@if ($paginator->hasPages())
    <nav role="navigation">
        {{-- Mobile --}}
        <div class="flex justify-between sm:hidden">
            @if ($paginator->onFirstPage())
                <span>← Anterior</span>
            @else
                <a href="{{ $paginator->previousPageUrl() }}">← Anterior</a>
            @endif

            @if ($paginator->hasMorePages())
                <a href="{{ $paginator->nextPageUrl() }}">Siguiente →</a>
            @else
                <span>Siguiente →</span>
            @endif
        </div>

        {{-- Desktop --}}
        <div class="hidden sm:flex">
            <span class="relative z-0 inline-flex shadow-sm rounded-xl">
                {{-- Anterior --}}
                @if ($paginator->onFirstPage())
                    <span aria-disabled="true">←</span>
                @else
                    <a href="{{ $paginator->previousPageUrl() }}">←</a>
                @endif

                {{-- Páginas --}}
                @foreach ($elements as $element)
                    @if (is_string($element))
                        <span>{{ $element }}</span>
                    @endif

                    @if (is_array($element))
                        @foreach ($element as $page => $url)
                            @if ($page == $paginator->currentPage())
                                <span class="bg-medico-azul-500 text-white">{{ $page }}</span>
                            @else
                                <a href="{{ $url }}">{{ $page }}</a>
                            @endif
                        @endforeach
                    @endif
                @endforeach

                {{-- Siguiente --}}
                @if ($paginator->hasMorePages())
                    <a href="{{ $paginator->nextPageUrl() }}">→</a>
                @else
                    <span aria-disabled="true">→</span>
                @endif
            </span>
        </div>
    </nav>
@endif

🎉 ¿Qué aprendimos hoy?

  • Laravel busca vistas de paginación en resources/views/vendor/pagination
  • La paginación tiene versión mobile (simple) y desktop (completa)
  • $paginator->currentPage() identifica la página activa para destacarla
  • Los colores médicos (medico-azul-*) mantienen la identidad visual
  • withQueryString() preserva filtros al cambiar de página
