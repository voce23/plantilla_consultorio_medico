Lección 6.4: Grid de Horarios Disponibles

Componente Livewire HorarioSelector - Toggle de horarios

🎯 Piensa en esto: Es como cuando vas a comprar entradas de cine. Ves una cuadrícula con todos los horarios, los que ya están vendidos aparecen tachados, y tú haces clic en el que quieres. Así funciona nuestro selector de horarios médicos.
1 Crear el componente Livewire

Este componente mostrará los horarios disponibles según el doctor y fecha seleccionados.

📍 Archivo: app/Livewire/HorarioSelector.php

class HorarioSelector extends Component
{
    public ?int $doctorId = null;
    public ?string $fecha = null;
    public ?string $horaSeleccionada = null;
    public array $horarios = [];

    public function mount(?int $doctorId = null, ?string $fecha = null)
    {
        $this->doctorId = $doctorId;
        $this->fecha = $fecha ?? now()->format('Y-m-d');
        $this->generarHorarios();
    }
}

💡 Las propiedades public son reactivas. Si cambian, la vista se actualiza sola.

2 Generar los slots de horario

Creamos intervalos de 30 minutos y verificamos cuáles están ocupados.

📍 Archivo: app/Livewire/HorarioSelector.php

public function generarHorarios()
{
    $this->horarios = [];
    
    // Horario de atención: 9:00 AM a 6:00 PM
    $inicio = Carbon::createFromTime(9, 0);
    $fin = Carbon::createFromTime(18, 0);

    // Obtener citas existentes
    $citasOcupadas = Cita::where('doctor_id', $this->doctorId)
        ->where('fecha', $this->fecha)
        ->whereIn('estado', ['pendiente', 'confirmada'])
        ->pluck('hora')
        ->map(fn ($hora) => Carbon::parse($hora)->format('H:i'))
        ->toArray();

🔍 Vamos a entender línea por línea:

wire:click="seleccionarHora('{{ $slot['hora'] }}')"
→ Cuando haces clic, Livewire ejecuta el método seleccionarHora en el servidor con la hora elegida.
@disabled(!$slot['disponible'])
→ Si el slot está ocupado, desactiva el botón. El usuario no puede hacer clic en horarios ya reservados.
$horaSeleccionada === $slot['hora'] ? 'bg-medico-azul-500' : ...
→ Operador ternario: si la hora está seleccionada, fondo azul; si no, verifica si está disponible.

📁 Método generarHorarios completo 
public function generarHorarios()
{
    $this->horarios = [];

    if (!$this->doctorId || !$this->fecha) {
        return;
    }

    $inicio = Carbon::createFromTime(9, 0);
    $fin = Carbon::createFromTime(18, 0);

    $citasOcupadas = Cita::where('doctor_id', $this->doctorId)
        ->where('fecha', $this->fecha)
        ->whereIn('estado', ['pendiente', 'confirmada'])
        ->pluck('hora')
        ->map(fn ($hora) => Carbon::parse($hora)->format('H:i'))
        ->toArray();

    while ($inicio <= $fin) {
        $hora = $inicio->format('H:i');
        $this->horarios[] = [
            'hora' => $hora,
            'disponible' => !in_array($hora, $citasOcupadas),
            'formato' => $inicio->format('h:i A'),
        ];
        $inicio->addMinutes(30);
    }
}

🎉 ¿Qué aprendimos hoy?

  • wire:click conecta clics del usuario con métodos PHP
  • @disabled desactiva botones condicionalmente
  • Carbon::createFromTime() crea objetos de fecha/hora
  • addMinutes(30) genera intervalos de media hora
  • Verificamos citas existentes para marcar horarios ocupados
