Lección 6.6: Integración WhatsApp

Botón dinámico con https://wa.me/ + datos del formulario

🎯 Piensa en esto: Es como tener un botón mágico que, al hacer clic, abre WhatsApp con un mensaje pre-escrito con todos los datos de tu cita. El paciente solo tiene que dar "Enviar". ¡Cero fricción!
1 Crear el componente del botón

Vamos a hacer un componente reutilizable que genere enlaces de WhatsApp dinámicos.

📍 Archivo: resources/views/components/boton-whatsapp.blade.php

@props([
    'telefono' => null,
    'mensaje' => null,
    'cita' => null,
    'tipo' => 'boton', // 'boton', 'flotante', 'link'
    'texto' => 'Contactar por WhatsApp',
])

💡 El componente acepta un array cita y automáticamente genera un mensaje profesional.

2 Generar el mensaje automáticamente

Si pasamos los datos de la cita, el componente crea un mensaje formateado con emojis.

📍 Archivo: resources/views/components/boton-whatsapp.blade.php

@php
if ($cita && !$mensaje) {
    $mensaje = "¡Hola! 👋\n\n";
    $mensaje .= "Solicito información sobre una cita médica:\n\n";
    
    if (isset($cita['paciente_nombre'])) {
        $mensaje .= "👤 *Paciente:* " . $cita['paciente_nombre'] . "\n";
    }
    
    if (isset($cita['doctor_nombre'])) {
        $mensaje .= "🩺 *Doctor:* " . $cita['doctor_nombre'] . "\n";
    }
    
    if (isset($cita['fecha'])) {
        $mensaje .= "📅 *Fecha:* " . $cita['fecha'] . "\n";
    }
    
    $mensaje .= "\n¿Podrían confirmarme? ¡Gracias! 🙏";
}

📝 ESTE ES EL CÓDIGO QUE VAMOS A APRENDER:

📍 Archivo: resources/views/components/boton-whatsapp.blade.php

@php
// Limpiar el número (quitar espacios, guiones, etc.)
$numeroLimpio = preg_replace('/[^0-9]/', '', $telefono);

// Codificar el mensaje para URL
$mensajeCodificado = urlencode($mensaje);

// URL de WhatsApp
$urlWhatsApp = "https://wa.me/{$numeroLimpio}?text={$mensajeCodificado}";
@endphp

<a href="{{ $urlWhatsApp }}" target="_blank" rel="noopener noreferrer">
    {{ $texto }}
</a>

🔍 Vamos a entender línea por línea:

preg_replace('/[^0-9]/', '', $telefono)
→ Expresión regular que elimina todo lo que no sea número. "+51 999-888-777" → "51999888777". WhatsApp necesita el número limpio.
urlencode($mensaje)
→ Convierte caracteres especiales a formato URL. Los espacios se vuelven %20, los saltos de línea %0A, etc.
https://wa.me/{$numero}?text={$mensaje}
→ URL oficial de WhatsApp Click to Chat. Al abrirla, WhatsApp web o app se abre con el mensaje listo.
target="_blank" rel="noopener noreferrer"
→ Abre en nueva pestaña de forma segura. noopener evita que la página nueva acceda a la ventana original.

📁 Ejemplo de uso del componente 
<!-- Botón estándar -->
<x-boton-whatsapp 
    telefono="+51 999 888 777"
    texto="Enviar confirmación"
    :cita="[
        'paciente_nombre' => 'Juan Pérez',
        'doctor_nombre' => 'Dr. García',
        'fecha' => '25/12/2024',
        'hora' => '10:00 AM',
    ]"
/>

<!-- Botón flotante -->
<x-boton-whatsapp 
    tipo="flotante"
    telefono="+51 999 888 777"
/>

<!-- Link simple -->
<x-boton-whatsapp 
    tipo="link"
    telefono="+51 999 888 777"
    texto="Contáctanos por WhatsApp"
/>

🎉 ¿Qué aprendimos hoy?

  • https://wa.me/ es la URL oficial de WhatsApp Click to Chat
  • preg_replace('/[^0-9]/', '', $numero) limpia el número de teléfono
  • urlencode() convierte texto a formato URL seguro
  • Podemos generar mensajes automáticos con datos de la cita
  • Tres tipos de botón: estándar, flotante y link
