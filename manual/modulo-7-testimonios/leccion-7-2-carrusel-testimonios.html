Lección 7.2: Carrusel de Testimonios

Componente Livewire + Alpine.js con transiciones suaves

🎯 Piensa en esto: Imagina un expositor de fotos en una mesa, donde tienes varias fotos de momentos especiales. En lugar de verlas todas amontonadas, las pones en un álbum que va pasando automáticamente de una foto a otra. Eso es exactamente lo que hace un carrusel: muestra una cosa a la vez y permite navegar entre ellas.
1 Crear el componente Livewire

Vamos a crear un componente que se encargue de manejar la lógica del carrusel: qué testimonio mostrar, pasar al siguiente, etc.

📍 Terminal:

php artisan make:livewire CarruselTestimonios

💡 Esto crea dos archivos: la clase PHP y la vista Blade.

2 La lógica del carrusel en PHP

Aquí definimos qué testimonios mostrar y las funciones para navegar entre ellos. Fíjate que usamos los scopes del modelo.

📍 Archivo: app/Livewire/CarruselTestimonios.php

public int $testimonioActual = 0;
public array $testimonios = [];

public function mount(int $cantidad = 6)
{
    $this->testimonios = Testimonio::aprobados()
        ->destacados()
        ->recientes()
        ->limit($cantidad)
        ->get()
        ->toArray();
}

public function siguiente()
{
    $this->testimonioActual = 
        ($this->testimonioActual + 1) % count($this->testimonios);
}

💡 El operador % (módulo) hace que cuando llegues al final, vuelvas al principio. Es como un círculo.

📝 ESTE ES EL CÓDIGO QUE VAMOS A APRENDER:

📍 Archivo: resources/views/livewire/carrusel-testimonios.blade.php

<div
    wire:key="testimonio-{{ $index }}"
    x-show="$wire.testimonioActual === {{ $index }}"
    x-transition:enter="transition ease-out duration-500"
    x-transition:enter-start="opacity-0 transform scale-95"
    x-transition:enter-end="opacity-100 transform scale-100"
    x-transition:leave="transition ease-in duration-300"
    x-transition:leave-start="opacity-100 transform scale-100"
    x-transition:leave-end="opacity-0 transform scale-95"
>
    <!-- Contenido del testimonio -->
</div>

🔍 Vamos a entender línea por línea (como si fuera la primera vez):

wire:key="testimonio-{{ $index }}"
→ Le damos una identificación única a cada testimonio. Es como ponerle un número de serie para que Livewire sepa cuál es cuál.
x-show="$wire.testimonioActual === {{ $index }}"
→ Solo mostramos este testimonio si su número coincide con el testimonio actual. Los demás se ocultan automáticamente.
x-transition:enter="transition ease-out duration-500"
→ Cuando el testimonio aparece, tarda 500 milisegundos (medio segundo) en hacerse visible. La animación es suave al final (ease-out).
x-transition:enter-start="opacity-0 transform scale-95"
→ Al inicio de la animación, el testimonio está invisible (opacity-0) y un poquito más pequeño (scale-95).
x-transition:enter-end="opacity-100 transform scale-100"
→ Al final de la animación, el testimonio es completamente visible (opacity-100) y a su tamaño normal (scale-100).
x-transition:leave="transition ease-in duration-300"
→ Cuando el testimonio se va, la animación dura 300 milisegundos y es suave al inicio (ease-in).
3 Los botones de navegación

Necesitamos flechas para que el usuario pueda cambiar de testimonio manualmente.

📍 Archivo: resources/views/livewire/carrusel-testimonios.blade.php

<button
    type="button"
    wire:click="anterior"
    class="absolute left-0 top-1/2 transform -translate-y-1/2 ..."
>
    <svg class="w-6 h-6" fill="none" stroke="currentColor">
        <path d="M15 19l-7-7 7-7"/>
    </svg>
</button>

💡 wire:click conecta el botón con la función del componente Livewire.

4 Los indicadores (dots)

Esos puntitos de abajo que indican cuántos testimonios hay y cuál estás viendo.

📍 Archivo: resources/views/livewire/carrusel-testimonios.blade.php

<div class="flex justify-center gap-2 mt-8">
    @foreach($testimonios as $index => $testimonio)
        <button
            wire:click="irA({{ $index }})"
            class="w-3 h-3 rounded-full transition-all duration-300 
                {{ $testimonioActual === $index ? 'bg-medico-azul-500 w-8' : 'bg-gray-300' }}"
        ></button>
    @endforeach
</div>

💡 Fíjate que el dot activo es más ancho (w-8) y de color azul, los demás son grises.

📁 ¿Quieres ver el código completo del componente? (opcional)

Esto es solo para que veas dónde va tu código. No necesitas memorizar todo esto:

<?php

namespace App\Livewire;

use Livewire\Component;
use App\Models\Testimonio;

class CarruselTestimonios extends Component
{
    public int $testimonioActual = 0;
    public array $testimonios = [];
    public bool $autoPlay = true;
    public int $intervalo = 5000;

    public function mount(int $cantidad = 6, bool $autoPlay = true)
    {
        $this->autoPlay = $autoPlay;
        $this->testimonios = Testimonio::aprobados()
            ->destacados()
            ->recientes()
            ->limit($cantidad)
            ->get()
            ->toArray();
    }

    public function siguiente()
    {
        $this->testimonioActual = 
            ($this->testimonioActual + 1) % count($this->testimonios);
    }

    public function anterior()
    {
        $this->testimonioActual = 
            ($this->testimonioActual - 1 + count($this->testimonios)) 
            % count($this->testimonios);
    }

    public function irA(int $index)
    {
        $this->testimonioActual = $index;
    }

    public function render()
    {
        return view('livewire.carrusel-testimonios');
    }
}

🎉 ¿Qué aprendimos hoy?

  • Livewire maneja la lógica (qué testimonio mostrar) y Alpine.js maneja las animaciones
  • x-transition crea animaciones suaves cuando los elementos aparecen y desaparecen
  • El operador módulo (%) permite que el carrusel sea circular: del último vuelve al primero
  • Los indicadores (dots) dan feedback visual sobre la posición actual
  • wire:click conecta los botones de la vista con las funciones PHP
