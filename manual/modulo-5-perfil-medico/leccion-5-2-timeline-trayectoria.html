Lección 5.2: Timeline de Trayectoria

Componente Livewire Timeline - Estudios y especializaciones

🎯 Piensa en esto: ¿Alguna vez has visto esos árboles genealógicos donde cada generación está conectada con líneas? Un timeline es igual, pero en lugar de ancestros, muestra los logros de una persona en orden cronológico. Es como contar la historia de tu vida profesional paso a paso.
1 Crear el componente Livewire

Livewire es como Blade con superpoderes: puede reaccionar a eventos del usuario sin escribir JavaScript. Vamos a crear uno para el timeline.

📍 Archivo: app/Livewire/TimelineTrayectoria.php

# Creamos la carpeta primero (si no existe)
mkdir app/Livewire

💡 En Windows puedes crear la carpeta desde el explorador de archivos o con PowerShell.

2 Definir la propiedad de eventos

En Livewire, las variables que empiezan con public son "reactivas". Si cambian, la vista se actualiza automáticamente.

📍 Archivo: app/Livewire/TimelineTrayectoria.php

<?php

namespace App\Livewire;

use Livewire\Component;

class TimelineTrayectoria extends Component
{
    public array $eventos = [];

    public function mount(array $eventos = null)
    {
        // Si no pasamos eventos, usar datos de ejemplo
        $this->eventos = $eventos ?? [
            ['anio' => '2024', 'titulo' => 'Director Médico', ...],
            ['anio' => '2020', 'titulo' => 'Especialización', ...],
        ];
    }

    public function render()
    {
        return view('livewire.timeline-trayectoria');
    }
}

💡 mount() es como el constructor de Livewire. Se ejecuta cuando el componente se inicializa.

3 Crear la vista del timeline

Ahora vamos a hacer que esos datos se vean bonitos. Usaremos una línea vertical con círculos para cada año.

📍 Archivo: resources/views/livewire/timeline-trayectoria.blade.php

# Crear la carpeta de vistas Livewire
mkdir resources/views/livewire

📝 ESTE ES EL CÓDIGO QUE VAMOS A APRENDER:

📍 Archivo: resources/views/livewire/timeline-trayectoria.blade.php

<!-- Línea vertical central -->
<div class="absolute left-8 md:left-1/2 transform md:-translate-x-px 
            top-0 bottom-0 w-0.5 
            bg-gradient-to-b from-medico-azul-500 via-medico-verde-500 to-medico-azul-200">
</div>

<!-- Cada evento del timeline -->
@foreach($eventos as $index => $evento)
    <div class="relative mb-12"
         x-intersect="visible[{{ $index }}] = true">
        
        <!-- Círculo con el año -->
        <div class="absolute left-8 md:left-1/2 transform -translate-x-1/2 
                    w-16 h-16 rounded-full flex items-center justify-center 
                    text-white font-bold shadow-lg z-10
                    bg-gradient-to-br from-medico-azul-500 to-medico-azul-600">
            {{ $evento['anio'] }}
        </div>
        
        <!-- Card con información -->
        <div class="p-6 bg-white rounded-2xl shadow-lg">
            <h3>{{ $evento['titulo'] }}</h3>
            <p>{{ $evento['institucion'] }}</p>
        </div>
    </div>
@endforeach

🔍 Vamos a entender línea por línea (como si fuera la primera vez):

public array $eventos = [];
→ Declaramos una variable pública que Livewire vigilará. array es el tipo de dato (una lista de elementos). El = [] la inicializa vacía.
$this->eventos = $eventos ?? [...]
→ El operador ?? significa "si no existe, usa esto". Es como decir: "¿Me pasaron eventos? No, entonces uso estos de ejemplo".
x-intersect="visible[{{ $index }}] = true"
→ x-intersect es de Alpine.js. Detecta cuando un elemento entra en la pantalla visible. {{ $index }} es el número del evento (0, 1, 2...).
md:left-1/2 transform -translate-x-1/2
→ En pantallas medianas, posiciona el elemento en la mitad (left-1/2) y luego lo mueve a la izquierda la mitad de su ancho (-translate-x-1/2) para centrarlo perfecto.

📊 Estructura de cada evento del timeline:

Clave Valor ejemplo Para qué sirve
anio '2024' Año del evento (aparece en el círculo)
titulo 'Director Médico' Nombre del logro o posición
institucion 'Clínica Premium' Donde ocurrió el evento
descripcion 'Liderazgo del equipo...' Detalles adicionales
tipo 'cargo', 'estudio', 'titulo' Define el color del círculo

👆 El campo "tipo" determina el color: azul para títulos, verde para estudios, gris para cargos.

📁 ¿Quieres ver el código completo del componente PHP? (opcional)

Este es el archivo completo del componente Livewire:

<?php

namespace App\Livewire;

use Livewire\Component;

class TimelineTrayectoria extends Component
{
    public array $eventos = [];

    public function mount(array $eventos = null)
    {
        $this->eventos = $eventos ?? [
            [
                'anio' => '2024',
                'titulo' => 'Director Médico',
                'institucion' => 'Clínica Médica Premium',
                'descripcion' => 'Liderazgo del equipo médico',
                'tipo' => 'cargo',
            ],
            [
                'anio' => '2020',
                'titulo' => 'Especialización en Cardiología',
                'institucion' => 'Universidad Nacional',
                'descripcion' => 'Estudios de posgrado',
                'tipo' => 'estudio',
            ],
            [
                'anio' => '2010',
                'titulo' => 'Médico Titulado',
                'institucion' => 'Universidad Cayetano Heredia',
                'descripcion' => 'Título profesional',
                'tipo' => 'titulo',
            ],
        ];
    }

    public function render()
    {
        return view('livewire.timeline-trayectoria');
    }
}

🎉 ¿Qué aprendimos hoy?

  • Livewire crea componentes interactivos sin escribir JavaScript
  • mount() inicializa datos cuando el componente carga
  • Las variables public en Livewire son reactivas
  • x-intersect de Alpine.js detecta cuando algo aparece en pantalla
  • El operador ?? proporciona valores por defecto
