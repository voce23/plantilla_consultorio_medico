Lección 5.1: Sección About con Blade

Componente DoctorProfile - Foto con clip-path y biografía

🎯 Piensa en esto: Imagina que vas a una entrevista de trabajo. ¿Qué es lo primero que ves en la pared del entrevistador? Un diploma con tu foto, tu nombre y tus logros más importantes. La sección "About" es exactamente eso: una tarjeta de presentación elegante que dice "¡Hola! Soy el doctor y esto es lo que hago".
1 Crear el archivo del componente

En Laravel, los componentes Blade viven en una carpeta especial. Es como tener un cajón organizado solo para tus piezas reutilizables.

📍 Archivo: resources/views/components/doctor-profile.blade.php

# No necesitas comando, creamos el archivo directamente

💡 Este componente lo vamos a usar en la página principal para mostrar la info del médico.

2 Definir las props del componente

Las "props" son como los ingredientes que necesita tu receta. En este caso: el doctor y si queremos mostrar estadísticas.

📍 Archivo: resources/views/components/doctor-profile.blade.php

@props([
    'doctor' => null,
    'showStats' => true
])

💡 doctor => null significa que si no pasamos un doctor, la variable será null y mostraremos datos de ejemplo.

3 Crear la estructura con grid de dos columnas

Vamos a dividir la pantalla en dos: izquierda para la foto, derecha para la información. Como un folleto médico profesional.

📍 Archivo: resources/views/components/doctor-profile.blade.php

<section class="py-20 bg-clinico-gris">
    <div class="px-4 mx-auto max-w-7xl">
        <div class="grid grid-cols-1 gap-12 lg:grid-cols-2 items-center">
            


    
📝 ESTE ES EL CÓDIGO QUE VAMOS A APRENDER:

    
📍 Archivo: resources/views/components/doctor-profile.blade.php

    
<!-- Foto con clip-path personalizado -->
<div class="relative overflow-hidden rounded-2xl" 
     style="clip-path: polygon(0 0, 100% 0, 100% 85%, 85% 100%, 0 100%);">
    @if($doctor && $doctor->foto)
        <img src="{{ asset('storage/' . $doctor->foto) }}" 
             alt="{{ $doctor->nombre_completo }}"
             class="w-full h-[400px] object-cover">
    @else
        <!-- Placeholder cuando no hay foto -->
        <div class="w-full h-[400px] bg-gradient-to-br from-medico-azul-200 to-medico-verde-200">
            <!-- SVG de usuario -->
        </div>
    @endif
</div>






🔍 Vamos a entender línea por línea (como si fuera la primera vez):




    <div style="clip-path: polygon(...)">
    
→ clip-path es como una tijera que recorta la imagen. polygon define los puntos del recorte. Los números son coordenadas: 0,0 es esquina superior izquierda, 100%,100% es esquina inferior derecha.





    @if($doctor && $doctor->foto)
    
→ Blade usa @if para preguntar "¿existe el doctor Y tiene foto?". Es como decir "si hay foto, muéstrala".





    {{ asset('storage/' . $doctor->foto) }}
    
→ asset() genera la URL completa de un archivo. El punto (.) une textos: "storage/" + nombre del archivo.





    object-cover
    
→ Clase de Tailwind que hace que la imagen cubra todo el espacio sin deformarse. Como cuando ajustas una foto de perfil en WhatsApp.






    
📊 Así se ve la estructura del componente:

    

        

            Prop
            Tipo
            Valor por defecto
            Qué hace
        

        

            doctor
            Model|null
            null
            El modelo Doctor con todos sus datos
        

        

            showStats
            boolean
            true
            Muestra/oculta las estadísticas rápidas
        

    

    
👆 Si doctor es null, el componente muestra datos de ejemplo para que la página no se vea vacía.







    📁 ¿Quieres ver el código completo del archivo? (opcional)
    
Esto es solo para que veas dónde va tu código. No necesitas memorizar todo esto:

    
<!-- Componente Doctor Profile - Sección About -->
<!-- 📁 Archivo: resources/views/components/doctor-profile.blade.php -->

@props([
    'doctor' => null,
    'showStats' => true
])

<section class="py-20 bg-clinico-gris overflow-hidden">
    <div class="px-4 mx-auto max-w-7xl sm:px-6 lg:px-8">
        <div class="grid grid-cols-1 gap-12 lg:grid-cols-2 lg:gap-16 items-center">
            
            <!-- Columna izquierda: Foto del doctor -->
            <div class="relative" x-data="{ show: false }" x-intersect="show = true">
                <!-- Decoración de fondo -->
                <div class="absolute inset-0 bg-gradient-to-br from-medico-azul-500 to-medico-verde-500 rounded-3xl transform rotate-3 scale-105 opacity-10"></div>
                
                <!-- Foto con clip-path -->
                <div class="relative bg-white p-3 rounded-3xl shadow-2xl">
                    <div class="relative overflow-hidden rounded-2xl" 
                         style="clip-path: polygon(0 0, 100% 0, 100% 85%, 85% 100%, 0 100%);">
                        @if($doctor && $doctor->foto)
                            <img src="{{ asset('storage/' . $doctor->foto) }}" 
                                 alt="{{ $doctor->nombre_completo }}"
                                 class="w-full h-[400px] lg:h-[500px] object-cover">
                        @else
                            <div class="w-full h-[400px] lg:h-[500px] bg-gradient-to-br from-medico-azul-200 to-medico-verde-200 flex items-center justify-center">
                                <div class="text-center">
                                    <svg class="w-32 h-32 mx-auto text-white opacity-80" fill="currentColor" viewBox="0 0 24 24">
                                        <path d="M12 12c2.21 0 4-1.79 4-4s-1.79-4-4-4-4 1.79-4 4 1.79 4 4 4zm0 2c-2.67 0-8 1.34-8 4v2h16v-2c0-2.66-5.33-4-8-4z"/>
                                    </svg>
                                    <p class="mt-4 text-white font-medium">Dr(a). {{ $doctor->nombre_completo ?? 'Médico' }}</p>
                                </div>
                            </div>
                        @endif
                    </div>
                </div>
            </div>

            <!-- Columna derecha: Información -->
            <div x-data="{ show: false }" x-intersect="show = true">
                <!-- Badge, título, biografía, estadísticas, etc. -->
            </div>
        </div>
    </div>
</section>

    • 
        
  • La directiva @if nos permite mostrar contenido condicional
    • 
        
  • asset() genera URLs correctas para archivos del proyecto
    • 
        
  • Grid de Tailwind divide la página en columnas responsivas
