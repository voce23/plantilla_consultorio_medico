🏥 Plantilla Consultorio Médico

Plantilla web gratuita y completa para consultorios médicos, clínicas
y profesionales de la salud. Lista para usar, personalizar y publicar.

🌐 Ver la demo en vivo

👉 [clinica.aulacursos.net](https://clinica.aulacursos.net/)


✅ ¿Qué incluye?

- Página de inicio con hero, estadísticas y llamada a la acción
- Listado de especialidades médicas
- Sistema de reserva de citas
- Página de contacto
- Diseño responsive (funciona en celular y computadora)
- Diseño moderno, limpio y profesional


🚀 ¿Cómo usarla?

1. Descarga o clona este repositorio
2. Abrí los archivos con tu editor (VS Code recomendado)
3. Reemplaza el nombre de la clínica, fotos y textos con los tuyos
4. Subí los archivos a tu hosting

No necesitas saber programar para personalizar los textos básicos.
